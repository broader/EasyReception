Downloading and installing Karrigell

Karrigell works with Python ; so if don't have Python installed on your computer, go to the Python website and download and install the latest version

The next step is obviously to download Karrigell. The package is available on sourceforge here. In "Latest file releases", follow the link "download". You will find three files, with the extensions zip, tgz or tar.bz2. Choose one (if you don't know, choose zip) and download the file

Unzip it with the adapted program, in a directory such as c:\Karrigell on Windows or /home/karrigell on Unix/Linux. The package will be in a subdirectory with the same name as the file you have downloaded (c:\Karrigell\Karrigell-version_number or /home/karrigell/Karrigell-version_number for instance)

In this directory you will see a module called kirbybase.py : KirbyBase is used as the default database engine on Karrigell. It is a fast, pure-Python flat-file database engine

You are now ready to start Karrigell : type python Karrigell.py

Karrigell 2.2 running on port 80
Debug level 1
Press Ctrl+C to stop

Note : the message may be different with other versions

This will start the server on the port 80, which is the default port for HTTP. If you want to use another port (for instance on a Unix / Linux machine where you don't have root privilege), specify it on the command line, like this :

python Karrigell.py -P 8080

Open a web browser and type http://localhost in the address field. You should see this page :

This home page provides link to see many demos and examples ; try the wiki, the "kwiki" which is the same as a wiki but with a wysiwyg editor, the forum, calendar, etc. In the "tour" you will be able to see the source code, this will give you a first idea of the variety of coding styles allowed by Karrigell
